# Manage Navigation

The Manage Navigation module controls the **entire website menu system**, including header navigation and footer widgets.

All changes made here are applied globally across the website.

---

## 🧭 Overview

This module is responsible for:

- Header navigation links  
- Dropdown menu structure  
- Footer widget groups (columns)  
- Linking pages to navigation system  

👉 Any page created in the CMS automatically becomes available here for linking.

---

## 🧩 Header Navigation

The header navigation controls the top menu of the website.

### Features:

- Add links to any CMS page  
- Connect navigation items directly to created pages  
- Support dropdown menus (multi-level navigation)  

### Dropdown Support

Users can create nested menu items:

- Main Menu Item  
  - Sub Item 1  
  - Sub Item 2  

This creates a dropdown in the frontend header.

## 🦶 Footer Navigation (Widget Groups)

Footer navigation is managed using **widget groups (columns)**.

### Structure:

Each footer group contains:

- Group Title (column heading)
- Multiple child links

Each child link includes:

- Label → Display text  
- URL → Destination link  

---

### Example:

**Company**
- About Us
- Contact
- Careers

**Support**
- Help Center
- Privacy Policy
- Terms

---

## 🧭 Page Linking System

All CMS pages automatically appear in the navigation selector.

Users can:

- Select any page from the CMS  
- Link it to header menu  
- Add it to footer widgets  
- Reuse links across multiple menus  

---

## 🔄 Update Behavior

- Changes are applied globally  
- Clicking **Update** instantly updates all pages  
- No manual code deployment required  
- Navigation is shared across entire website  

---

## ⚙️ System Behavior

- Header and footer navigation are stored centrally  
- All pages use the same navigation configuration  
- Dropdown structure supports multi-level menus  
- Pages are dynamically synced with navigation system  

---

## 🚀 Summary

Manage Navigation controls the full website menu system:

- Header menus with dropdown support  
- Footer widget-based navigation  
- Page linking system  
- Global update across all pages  

This ensures consistent navigation across the entire CMS website.

# Header Settings

The Header Settings module allows users to control the global header and footer layout behavior of the website.

These settings define how the navigation and footer appear across all pages.

---

## 🧭 Header Configuration

### Header Template

Select the global header layout used across the website.

- Choose from available header designs
- Applied site-wide
- Controls navigation structure and UI style

---

### Scroll Behavior

Define how the header behaves when scrolling:

- **Fixed** → Header stays always visible at the top  
- **Sticky** → Header becomes fixed after scrolling  
- **Default** → Standard scrolling behavior  

---

## 🦶 Footer Configuration

### Footer Template

Select the global footer layout used across the website.

- Choose from available footer designs  
- Applied to all pages  
- Controls layout and structure of footer sections  

---

## 🧠 System Behavior

- Header and footer settings are global  
- Changes apply immediately across the website  
- No code changes required  
- Affects all pages using default layout system  

---

## 🚀 Summary

Header Settings control the global layout behavior of:

- Website header design  
- Navigation style  
- Scroll behavior  
- Footer layout template  

This ensures consistent UI across the entire website.

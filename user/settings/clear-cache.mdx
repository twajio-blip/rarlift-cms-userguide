# Clear Cache

The Clear Cache module is used to refresh and reset the system cache of the CMS.

It ensures that all recent changes are properly reflected on the live website.

---

## 🧹 What Clear Cache Does

When executed, the system performs a full Laravel cache reset, including:

- Configuration Cache  
- Route Cache  
- View Cache  
- Application Cache  

---

## ⚡ One-Click Operation

Clear Cache is a **single-action tool**.

Users only need to click the **Clear Cache button**, and the system will automatically:

1. Clear all cached data  
2. Rebuild configuration files  
3. Refresh system routes  
4. Reload compiled views  

---

## 🧠 When to Use It

Use Clear Cache when:

- Changes in settings are not visible on the frontend  
- Updated components are not reflecting on the website  
- Navigation or layout changes are not applied  
- System behavior appears outdated  

---

## ⚠️ Important Note

- This action does not delete any data  
- It only clears temporary cached files  
- It is safe to use at any time  
- Recommended after major configuration changes  

---

## 🚀 Summary

Clear Cache ensures the CMS always reflects the latest updates by resetting all system-level caches with a single click.

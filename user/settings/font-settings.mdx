# Font Settings

The Font Settings module allows users to control typography across both the frontend website and the CMS admin panel.

It ensures consistent branding and readability across all interfaces.

---

## 🌐 Frontend Fonts (Website)

Frontend font settings control how the public-facing website appears to visitors.

### Font Link

- Add Google Fonts `<link>` embed code
- Used to load fonts from external sources (e.g. Google Fonts)

Example:
```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet">

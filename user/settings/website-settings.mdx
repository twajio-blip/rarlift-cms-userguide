# Website Settings

The Website Settings module controls the global configuration of the entire CMS website.

It includes branding, contact information, integrations, social links, footer structure, and logo management.

---

## 🧭 Settings Tabs Overview

The Website Settings page is divided into **4 tabs**:

- General
- Social Links
- Footer Links
- Logo

---

## ⚙️ General Tab

The General tab controls core website information and integrations.

### Available Fields:

- Website Name → Name shown in browser tab  
- Contact Number → Business phone number  
- Email → Official contact email  
- Location → Physical address  
- Description → Short company description  
- Google Analytics → Tracking ID (G-XXXXXXX)  
- Cookie Banner → Enable/disable cookie consent  
- reCAPTCHA → Site Key + Secret Key for form protection  

---

### 🍪 Cookie Settings

When enabled, a cookie consent banner appears on the frontend.

- Cookie Title  
- Cookie Description  
- Dismiss button enabled

## 🌐 Social Links Tab

The Social Links tab allows management of social media platforms.

### Features:

- Add or remove social links  
- Maximum recommended: 4 platforms  
- Each entry includes:

  - Name (e.g. Facebook)  
  - Icon (Font Awesome class like `fa-facebook`)  
  - URL (full link)  

### Example:

- Facebook  
- Instagram  
- Twitter  
- LinkedIn  

### Social Links Preview

![Social Links](/images/social-links.png)

---

## 🧩 Footer Links Tab

The Footer Links tab manages footer navigation structure.

### Features:

- Supports up to 2 groups (columns)
- Each group has:
  - Group Name (column title)
- Each group contains multiple links:
  - Link Name  
  - Icon (optional)  
  - URL  

### Example:

- Company
- Support

### Footer Links Preview

![Footer Links](/images/footer-links.png)

---

## 🖼️ Logo Tab

The Logo tab manages all branding assets of the website.

### Available Logos:

- Fav Icon → Browser tab icon  
- Header Logo → Logo shown in top navigation  
- Footer Logo → Logo shown in footer area  

---

### Logo Behavior

- Header and footer logos are controlled separately  
- Changes apply instantly across the website  
- Fav icon updates browser tab icon  

### Logo Settings Preview

![Logo Settings](/images/logo-settings.png)

---

## ⚙️ System Behavior

- All settings are global (affect entire website)  
- Updates are reflected immediately on frontend  
- No code changes required  
- Stored centrally in CMS database  

---

## 🚀 Summary

Website Settings control the entire identity and integration layer of the CMS:

- Branding  
- Contact information  
- Social media  
- Footer structure  
- Logos  
- Tracking and security tools  

This module ensures full control of website configuration without coding.

# Notifications

The Notifications section shows all incoming activity from the website, including customer messages and newsletter subscribers.

This system helps users track real-time engagement from visitors.

---

## 🔔 Notifications Overview

Notifications are automatically generated when users interact with the website.

Main sources:

- Contact form submissions (Inquiries)
---

## 📩 Inquiries (Contact Messages)

The Inquiries section displays messages sent from the website contact form.

Each inquiry includes:

- Sender Name  
- Email Address  
- Subject  
- Message Content  
- Submission Time  
- Status (Unread / Read / Replied)

### What Users See

Users can open each message to view full details and manage its status.

## 🔔 Topbar Notification System

The CMS includes a notification badge in the top navigation bar.

### It shows:

- Number of unread inquiries  
- Number of new subscribers  
- Latest activity preview  

### Behavior:

- Badge updates automatically when new activity occurs  
- Clicking opens the notification dropdown  
- Users can navigate to full details page

## ⚙️ Notification Behavior

- Notifications are generated automatically by the system  
- No manual input required  
- Data updates in real-time or page refresh  
- Old notifications remain accessible for records

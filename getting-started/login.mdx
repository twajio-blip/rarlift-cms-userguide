# Login

The RARLift CMS uses a secure authentication system. Only authorized users can access the CMS dashboard.

The system is designed with a single role model, meaning there are no separate roles like editors or guests.

---

## Who can access the CMS?

Only authenticated users can access the system.

- Public visitors → No access to CMS  
- CMS users → Full access to dashboard and management tools  

---

## Login credentials

Users must log in using:

- Email address  
- Password  

---

## Login process

To access the CMS:

1. Open the CMS login page  
2. Enter your email address  
3. Enter your password  
4. Click the Login button  

If the credentials are correct, you will be redirected to the dashboard.

## After login

Once authenticated, the user can:

- Manage website pages  
- Add and edit components  
- Update content such as text, images, and branding  
- Control page structure and layout  
- View system notifications  

---

## Security behavior

- Sessions are maintained after login  
- Unauthorized access to admin routes is blocked  
- Logout ends the session and redirects to the login page  

---

## System behavior

The authentication system ensures:

- Secure access control  
- Persistent user sessions  
- Protection of CMS routes  
- Clean logout handling
